위치 검색 앱
==========

## **기능 요구사항**

1. 오디오 및 비디오 파일 목록 표시
2. 선택한 미디어 재생
3. 재생/일시정지, 탐색 컨트롤
4. 볼륨 조절
5. 현재 재생 중인 미디어의 메타데이터 표시

## **UI 구성**

1. 미디어 목록 뷰
2. 재생 화면 (오디오/비디오)
3. 컨트롤 패널
